🔗 Apps Script Web App URL
Cole aqui a URL gerada ao implantar o fatburn_apps_script.js como Web App no Google Apps Script. Esta URL conecta o dashboard às 3 integrações simultaneamente.
📋 Como configurar (5 passos)
  • 1Abra script.google.com → "Novo projeto" → cole o conteúdo de fatburn_apps_script.js
  • 2Preencha as constantes CFG no topo: META_TOKEN, META_AD_ACCOUNT e SHEET_ID
  • 3Clique em "Executar" → rode setup() e depois testConnections()
  • 4Deploy → "Implantar como Web App" → Execute as: Me · Access: Anyone → Copie a URL
  • 5Adicione trigger: ícone de relógio → runDailySync → Time-driven → Day timer → 8h
